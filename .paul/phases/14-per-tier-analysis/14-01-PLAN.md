<objective>
## Goal
For each size tier (small, large) separately: compute hit rate, avg return, avg alpha on the matured sub-pool, run Fisher's exact vs baseline (142 mature strong_buy, 66.9% HR) and Mann-Whitney U on returns, apply Bonferroni correction (n=3 comparisons: small, large, and a combined "extended" tier). Produce a concrete adoption recommendation per tier for Phase 15.

## Purpose
Extend the statistical discipline from Phase 11 to the new tier candidates. We only extend `conviction_tier` values in Phase 15 if the per-tier sub-pool clears p<0.05 Bonferroni vs the existing 142 strong_buy baseline. If nothing clears, v1.5 closes without new tiers.
</objective>

## AC-2: Significance tests run with Bonferroni adjustment

```gherkin
Given per-tier statistics
When we compare each candidate tier (small, large) against the midcap baseline
Then Fisher's exact + Mann-Whitney U p-values are computed
  And Bonferroni-adjusted p-values use n=3 (midcap baseline as control, 2 tier comparisons + 1 combined "extended")
```

## AC-3: Recommendations are evidence-backed

```gherkin

When we read the Recommendations section

  And every recommendation cites its tier's n_mature, hit rate, p_fisher, and p_bonferroni
```
