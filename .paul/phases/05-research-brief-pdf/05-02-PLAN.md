    2. Create `backend/exports/brief_pdf.py`:

       **Imports:**
       ```python
       import argparse, re, sys
       from datetime import date
       from pathlib import Path
       import markdown
       from weasyprint import HTML, CSS
       ```

       **Core pipeline (one async-free function):**
       - Read `brief_v1_1.md` (default path: `../.paul/phases/05-research-brief-pdf/brief_v1_1.md`)
       - Convert to HTML via `markdown.markdown(src, extensions=['tables','toc','fenced_code','attr_list'])` — the `toc` extension is important for the TOC target in Task 2
       - Split the HTML into (a) cover content and (b) body content:
         - Cover = everything before the first `<h2>` (Executive Summary). This includes the title, subtitle, metadata table, and confidentiality line.
         - Body = from the first `<h2>` onward.
       - Wrap in a complete HTML document with the structure:
         ```html
         <!DOCTYPE html>
         <html>
           <head><meta charset="utf-8"><title>LookInsight — Insider Conviction Signals</title></head>
           <body>
             <section class="cover">{cover_html}</section>
             <section class="toc-page"><h2>Table of Contents</h2><div class="toc-placeholder"></div></section>
             <section class="body">{body_html}</section>
           </body>
         </html>
         ```
       - The `toc-placeholder` will be populated by CSS `target-counter` + generated content at print time (see CSS), or fallback: render an explicit TOC from headings parsed via regex.
       - Chart image paths in the markdown are relative (`charts/funnel.png`). weasyprint's HTML object needs an absolute base URL: pass `base_url=str(brief_md_path.parent.resolve())`.
       - Call `HTML(string=html, base_url=base_url).write_pdf(output_path, stylesheets=[CSS(filename=css_path)])`.

       **CLI:**
       - `--md <path>` (default: the canonical Phase 5 brief_v1_1.md)
       - `--css <path>` (default: backend/exports/brief.css)
       - `--output <path>` (default: the canonical PDF location in Phase 5)

       **Entry point:** `if __name__ == "__main__": sys.exit(main())`.

    3. Create `backend/exports/brief.css` — the institutional theme. Required elements (can be adjusted in Task 2, but must be present in Task 1):

       ```css
       /* Page geometry and running headers/footers */
       @page {
         size: letter;
         margin: 20mm 18mm 22mm 18mm;
         @bottom-center { content: "Page " counter(page) " of " counter(pages); font-size: 9pt; color: #666; }
         @top-center { content: "LookInsight — Insider Conviction Signals (v1.1)"; font-size: 9pt; color: #888; }
       }
       @page :first { @top-center { content: normal; } @bottom-center { content: normal; } }

       body {
         font-family: "Charter", "Source Serif Pro", "Georgia", serif;
         font-size: 10.5pt;
         line-height: 1.5;
         color: #222;
       }
       h1 { font-family: "Helvetica Neue", "Inter", sans-serif; font-size: 22pt; font-weight: 700; margin-top: 0; }
       h2 { font-family: "Helvetica Neue", "Inter", sans-serif; font-size: 16pt; font-weight: 600; border-bottom: 1px solid #ccc; padding-bottom: 4px; margin-top: 24pt; page-break-after: avoid; }
       h3 { font-family: "Helvetica Neue", "Inter", sans-serif; font-size: 12pt; font-weight: 600; margin-top: 14pt; page-break-after: avoid; }
       h4 { font-family: "Helvetica Neue", "Inter", sans-serif; font-size: 11pt; font-weight: 600; color: #444; margin-top: 10pt; page-break-after: avoid; }

       /* Cover page */
       section.cover { page-break-after: always; padding-top: 40mm; }
       section.cover h1 { font-size: 26pt; }
       section.cover table { width: 100%; margin-top: 16pt; border-collapse: collapse; }
       section.cover table td { padding: 6px 10px; border-top: 1px solid #eee; }
       section.cover em { color: #666; }

       /* Tables */
       table { width: 100%; border-collapse: collapse; font-size: 9.5pt; margin: 8pt 0; page-break-inside: auto; }
       thead { display: table-header-group; }
       th { background: #f0f0f0; border-bottom: 1.5px solid #888; text-align: left; padding: 4px 6px; font-weight: 600; }
       td { border-bottom: 1px solid #e8e8e8; padding: 3px 6px; }
       tr:nth-child(even) td { background: #fafafa; }

       /* Images */
       img { max-width: 100%; height: auto; display: block; margin: 8pt auto; }

       /* Avoid orphan/widow */
       p, li { orphans: 3; widows: 3; }
       ```
